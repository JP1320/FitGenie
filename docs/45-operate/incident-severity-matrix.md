# Incident Severity Matrix

- Sev1: Core journey broken (signup/booking/recommendation outage)
  - Response SLA: immediate
- Sev2: Major degradation (high failure/latency)
  - Response SLA: <30 min
- Sev3: Minor issue/workaround available
  - Response SLA: same day
- Sev4: Cosmetic/non-urgent
  - Response SLA: backlog
