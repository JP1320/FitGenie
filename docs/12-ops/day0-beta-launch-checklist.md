# Day 0 Beta Launch Checklist

## Release Control
- [ ] Release tag created (e.g. v0.2.0-beta.1)
- [ ] Changelog updated
- [ ] Rollback version identified
- [ ] Feature flags reviewed

## Environment
- [ ] Production env vars verified
- [ ] DB backup snapshot taken
- [ ] Readiness endpoint green (/ready)
- [ ] Error tracking active (Sentry or equivalent)

## Product
- [ ] Core journey verified end-to-end
- [ ] AI outputs validated against schema
- [ ] Order tracking statuses functioning
- [ ] Feedback submission functioning

## Go/No-Go
- [ ] PM sign-off
- [ ] Engineering sign-off
- [ ] Support sign-off
- [ ] Launch approved
