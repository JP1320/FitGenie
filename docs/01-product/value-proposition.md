# Value Proposition

## Core Promise

> **FitGenie gives every person the confidence of a perfect fit — whether buying ready-made, ordering custom stitching, or discovering a designer — in minutes, not days.**

---

## Value by User Segment

### For Customers

| Value | Description |
|---|---|
| **Right size, every time** | AI Fit Scanner measures body proportions from a smartphone camera and maps them to garment sizes, eliminating guesswork. |
| **Personalised recommendations** | The recommendation engine considers body type, style preference, fabric, budget, and occasion to suggest outfits that genuinely suit the user. |
| **Transparent confidence** | Every recommendation includes a Size Confidence Score and a human-readable explanation ("Why this suits you"), building trust in AI output. |
| **Seamless tailor access** | Users can discover, compare, and book tailors/designers directly from the recommendation screen, without leaving the app. |
| **Structured handoff** | The AI Fit Card auto-generates all measurements, preferences, and outfit notes — eliminating the back-and-forth between customer and tailor. |
| **End-to-end tracking** | From booking confirmation to final delivery or pickup, the order is tracked in real time. |

### For Tailors and Designers

| Value | Description |
|---|---|
| **Digital visibility** | Listing on FitGenie marketplace gives tailors discoverability beyond their local walk-in radius. |
| **Structured client data** | Fit Cards arrive with complete, machine-generated measurements and notes — reducing rework and misunderstandings. |
| **Appointment management** | Calendar-based booking eliminates phone tag; tailors control their own schedule. |
| **Portfolio showcase** | Designer profiles include portfolio images, ratings, specialisation, and price range. |
| **Feedback loop** | Post-delivery ratings and photos help tailors improve their service and build social proof. |

---

## Differentiation from Alternatives

| Alternative | Limitation | FitGenie Advantage |
|---|---|---|
| Standard e-commerce size charts | Brand-specific, inaccurate, no body context | AI scan + personalised mapping |
| Standalone tailoring apps | No AI, no recommendation, no marketplace | Full end-to-end journey in one app |
| Social fashion apps | Trend-driven, not fit-driven | Fit-first, then style |
| Visiting a tailor in person | Time-consuming, no digital record | Digital-first with same personal outcome |
| AI styling assistants (generic) | No body scan, no booking, no fit card | Integrated scan → recommend → book → deliver |

---

## Unique Selling Points (USPs)

1. **AI Fit Scanner** — camera-based, instant body measurement from a smartphone.
2. **Size Confidence Score** — quantified certainty for every size recommendation.
3. **AI Fit Card** — structured, shareable profile that bridges customer intent and tailor execution.
4. **"Why this suits you" explainability** — AI-generated, body-positive rationale for every recommendation.
5. **Integrated marketplace** — discover, compare, book, and track tailors/designers without leaving the app.

---

## Phase 1 Value Delivery (MVP Scope)

The Phase 1 build delivers:

- Body scan → measurement extraction → size recommendation
- Preference input → AI outfit recommendation with explanations
- Fit Card generation and sharing
- Tailor/designer discovery and booking
- Order status tracking

Phase 1 does **not** include:
- Real-time in-app chat (Phase 2)
- Payment gateway integration (Phase 2)
- Mobile-native app (Phase 1 targets responsive web; native apps in Phase 2)
