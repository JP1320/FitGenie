# Weekly Business Review (WBR)

## Reliability
- API success rate
- P95 latency
- Incident count by severity

## Funnel
- Signups
- Profile completion rate
- Recommendation CTR
- Booking conversion rate

## Quality
- Fit satisfaction score
- Return/alteration rate
- Tailor SLA compliance

## Financial
- Revenue
- Gross margin
- AI cost per successful booking

## Decisions
- Continue / stop experiments
- Priority fixes for next sprint
