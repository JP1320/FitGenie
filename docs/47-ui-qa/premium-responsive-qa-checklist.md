# Premium Responsive QA Checklist

## Breakpoints
- [ ] 375x812 (mobile)
- [ ] 768x1024 (tablet)
- [ ] 1440x900 (desktop)

## Visual
- [ ] No overflow/cutoff on cards or buttons
- [ ] Consistent spacing rhythm
- [ ] Typography hierarchy preserved

## Interaction
- [ ] Touch targets >= 44px on mobile
- [ ] Keyboard tab order logical
- [ ] Focus states visible

## States
- [ ] Loading skeleton present
- [ ] Empty state present
- [ ] Error state present
- [ ] Success toast present
