# Pricing Experiment Plan

## Objective
Increase booking conversion and average order value.

## Experiments
1. A/B test: "Free fit-card with first booking"
2. A/B test: "Priority tailor badge"
3. A/B test: "Bundle discount for repeat users"

## Metrics
- Booking conversion
- AOV (average order value)
- Repeat booking rate (30-day)
- Refund/alteration impact
